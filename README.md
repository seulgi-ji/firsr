# HTML
## HTML 구조태그
* HTML 은 **웹문서**를 뜻한다.
* `태그` 웹문서에 들어가는 내용~제목 모든 요소는 `<태그>`로 묶여서 만들어진다. 
## HTML 주의사항
* 파일명 . html 
* 모든 파일명, 폴더명은 영문 소문자, 특수문자 금지, 공백 금지, 한글 금지, 숫자 앞 금지
* start. html = O
* start1. html = O
* 1start. html = X(숫자앞)
* basic start. html = X(공백)
* basic_start.html = O(허용, 특수문자 _-)
* basic%start.html = X(허용 외 특수문자)
* 시작. html = X(한글)
## VSC 단축키
* 화면 분할(좌우) : `ctrl+\`
* 화면 분할(상하) : `ctrl+k` -> `ctrl+/`
* 찾기 - 바꾸기 : 바꾸고 싶은 글자 선택하고 -> `ctrl + h`
* 커서 위치 기준 한 줄 복제 : `shift + Alt + ↑↓`
* 커서 위치 기준 라인 이동 : `Alt + ↑↓`
## HTML 단축키

## 태그속성 class, id
### class
* `<태그 class="값"></태그>`
* 2번 이상 반복하는 이름인 경우
* 예) inb, snb, container 등
### id
* `<태그 id="값"></태그>`
* 반복하면 안되는 단독 명칭인 경우
* 예) header, gnb, footer 등
## HTML 기초 문서 태그
* 블록태그 : w100%, 새로운 행, 크기가 큼.
* 인라인태그 : 열(옆으로 나열), 크기는 내용만큼 인식.
### 제목
* `h1~h6` : 블록태그,h1대제목, h6소제목, h레벨이 높을수록 검색어로 인식될 확률높음(1~3), 필요시 4까지 쓰기도 함. SEO 마케팅 목적
### 내용
* `p` : 블록, 한줄~여러줄 단락형태의 내용일때
* `br` : 인라인, 줄바꿈할때 사용  
### 레이아웃
* `div` : 블록, 그룹, 자식이 2개 이상이며 같은 방향일때, 블록과 인라인을 자식으로 모두 가질 수 있음.
* `span` : 인라인, 그룹(위 div와 같음), 보조내용, 인라인만 자식으로 가질 수 있음.
